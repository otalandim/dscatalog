# CRUD - Bootcamp Spring 3.0 - DevSuperior DSCatalog

Criar projeto Spring Boot, criar monorepositório Git, organizar projeto em camadas, criar entidades, perfis de projeto, seeding da base de dados, criar web services REST, padrão DTO, CRUD completo, tratamento de exceções, dados de auditoria e paginação de dados. Você também vai aprender a trabalhar com Postman para testar as requisições.

## Tecnologias utilizadas

 - [Java 11](https://www.oracle.com/java/technologies/downloads/)
 - [Spring Boot](https://spring.io/)
 - [Maven](https://maven.apache.org/download.cgi)
 
 ## Dependências
 - Spring Web
 - Spring Data JPA
 - H2 Database
